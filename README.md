# 🚗 Car Game (C++ Console Project)

A simple **console-based car dodging game** built using C++.  
The player controls a car to avoid incoming enemy vehicles and survive as long as possible while earning points.

---

## 🎮 Features

- Smooth console-based gameplay  
- Moving enemy cars  
- Collision detection system  
- Score tracking system  
- Pause and resume functionality  
- Restart option without closing the game  
- Simple ASCII-style graphics  

---

## 🎯 Controls

- A → Move Left  
- D → Move Right  
- P → Pause / Resume  
- R → Restart Game  
- ESC → Exit Game  

---

## 🧠 Game Rules

- Avoid collision with enemy cars  
- Each successfully avoided enemy increases your score  
- Game ends when a collision occurs  
- You can pause or restart anytime during gameplay  

---

## 🛠️ Requirements

- Windows OS (uses `windows.h`)  
- C++ compiler (GCC / MinGW / MSVC)  

---

## 🚀 How to Run
---
### Compile the program
---
g++ Modified_car_game.cpp -o car_game
